## Description
This project provides a complete solution for setting up a Raspberry Pi (RPi) wall panel integrated with Home Assistant. It allows users to control their smart home devices, access voice assistant features, and much more.

## Features
- **Monitor Control**: Manage your smart home devices through a user-friendly interface.
- **Voice Assistant**: Integrate voice commands for seamless control.
- **Kiosk Mode**: Display your Home Assistant dashboard in a dedicated mode.
- **MQTT Integration**: Utilize MQTT for real-time communication with your devices.

## Hardware Requirements
| Component     | Description         |
|---------------|---------------------|
| RPi 4        | Raspberry Pi 4 Model |
| Monitor      | 15.6" HDMI Monitor   |
| USB Audio    | USB Audio Adapter    |
| Cables       | HDMI and power cables|

## Software Requirements
- **RPi OS Lite**: 64-bit version recommended
- **Home Assistant**: The core software for managing smart home devices
- **MQTT**: For message brokering between devices

## Quick Start Guide
1. **Flash OS**: Use Raspberry Pi Imager to flash the RPi OS Lite onto your microSD card.
2. **SSH Setup**: Enable SSH for remote access.
3. **Install Script**: Run the installation script provided in this repository.
4. **Hardware Connection**: Connect your monitor, USB audio, and power cables.
5. **HA Configuration**: Access Home Assistant and configure your devices.

## Voice Assistant Test Instructions
To test the voice assistant functionality, follow these steps:
- Ensure your USB audio is connected.
- Use the designated voice commands to interact with your devices.

## Monitor Control Test Instructions
- Access the monitor interface and use touch or mouse controls to navigate.
- Test various functionalities such as turning devices on/off.

## Automation Examples
- **Automatic Lighting**: Set up rules to turn on lights at sunset.
- **Temperature Control**: Adjust your heating based on the temperature readings.
